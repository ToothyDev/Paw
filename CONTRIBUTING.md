# Contributing to Paw

Thank you for wanting to contribute to Paw, and even reading this file!
The following is just a little help and legal stuff to make the project work.

# Pull Requests

To submit a pull request, fork this repository, make your changes on a separate branch, and when you are done,
open a pull request here, with a description of your changes and a fitting title. Lastly, request a review from me.
